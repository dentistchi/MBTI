# NEON MBTI UNIVERSE

MBTI와 혈액형 기반 성격 분석 웹사이트

## 📁 프로젝트 구조

```
neon-mbti-website/
├── index.html       # 메인 HTML 파일
├── styles.css       # 스타일시트 (네온 효과, 글래스 모피즘)
├── script.js        # JavaScript (차트, 언어 변경 기능)
└── README.md        # 프로젝트 설명서
```

## 🚀 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 또는 로컬 서버 실행:
   ```bash
   python -m http.server 8000
   ```
   그리고 `http://localhost:8000` 접속

## 🎨 주요 기능

- **네온 스타일 UI**: 보라색 네온 효과와 글래스모피즘 디자인
- **레이더 차트**: Chart.js를 이용한 역량 시각화
- **언어 변경**: 한국어/영어 전환 기능
- **반응형 디자인**: 모바일/데스크톱 대응

## 🛠 기술 스택

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (Vanilla JS)
- Chart.js
- Font Awesome Icons
